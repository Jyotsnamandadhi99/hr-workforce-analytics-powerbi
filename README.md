# HR Workforce Analytics Dashboard

## Overview

This project focuses on building an interactive HR Workforce Analytics Dashboard using Power BI. The dashboard aims to help HR professionals analyze workforce demographics, employee performance, recruitment effectiveness, attrition trends, attendance, and salary distribution through interactive visualizations.

## Objectives

- Analyze workforce demographics
- Monitor employee attrition
- Evaluate recruitment effectiveness
- Assess employee performance and engagement
- Explore salary distribution
- Identify attendance patterns
- Generate actionable HR insights

## Dataset

**Dataset Name:** HR-Dataset

The dataset contains **311 employee records** and **35 attributes**, including employee demographics, employment details, recruitment information, salary, performance, engagement, and attendance data.
