# HR Workforce Analytics Dashboard

## Overview

This project focuses on building an interactive HR Workforce Analytics Dashboard using Power BI. The dashboard provides actionable insights into workforce demographics, employee performance, recruitment, attrition, attendance, and salary trends to support HR decision-making.

## Objectives

- Analyze workforce demographics
- Monitor employee attrition
- Evaluate recruitment effectiveness
- Assess employee performance and engagement
- Explore salary distribution
- Identify attendance patterns
- Generate business insights through interactive dashboards

## Dataset

**Dataset:** HR-Dataset

The dataset contains employee demographic, employment, recruitment, salary, performance, engagement, and attendance information.
