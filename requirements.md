# Project Requirements

This document lists the software and tools required to run, modify, and explore the **HR Workforce Analytics Dashboard**.

---

# Software Requirements

| Software | Version |
|----------|---------|
| Windows | 10 / 11 |
| Power BI Desktop | Latest Version |
| Microsoft Excel | 2019 or later |
| Git | Latest Version |
| Visual Studio Code (Optional) | Latest Version |

# Hardware Requirements

## Minimum

- Dual-Core Processor
- 4 GB RAM
- 2 GB Free Storage
- 1366 × 768 Display

## Recommended

- Intel Core i5 / Ryzen 5 or higher
- 8 GB RAM or more
- SSD Storage
- 1920 × 1080 (Full HD) Display
