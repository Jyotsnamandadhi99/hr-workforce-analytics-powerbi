# Data Dictionary

## Dataset Information

**Dataset Name:** HR-Dataset

**Description:**
This dataset contains employee demographic, employment, performance, salary, recruitment, and attendance information. It is used to build an interactive HR Workforce Analytics Dashboard in Power BI.

| Column Name | Data Type | Description |
|------------|-----------|-------------|
| Employee_Name | Text | Full name of the employee. |
| EmpID | Integer | Unique identifier assigned to each employee. |
| MarriedID | Integer | Binary indicator representing marital status (0 = Not Married, 1 = Married). |
| MaritalStatusID | Integer | Numeric code representing marital status. |
| GenderID | Integer | Numeric code representing employee gender. |
| EmpStatusID | Integer | Numeric code representing employment status. |
| DeptID | Integer | Numeric identifier for the employee's department. |
| PerfScoreID | Integer | Numeric identifier for performance rating. |
| FromDiversityJobFairID | Integer | Indicates whether the employee was recruited through a diversity job fair (0 = No, 1 = Yes). |
| Salary | Currency | Annual salary of the employee. |
| Termd | Integer | Indicates whether the employee has been terminated (0 = Active, 1 = Terminated). |
| PositionID | Integer | Numeric identifier for the employee's job position. |
| Position | Text | Job title or designation of the employee. |
| State | Text | State where the employee is located. |
| Zip | Integer | Employee's ZIP/Postal code. |
| DOB | Date | Employee's date of birth. |
| MaritalDesc | Text | Marital status description. |
| CitizenDesc | Text | Citizenship status of the employee. |
| HispanicLatino | Text | Indicates Hispanic/Latino ethnicity. |
| RaceDesc | Text | Employee's race or ethnicity category. |
| DateofHire | Date | Date the employee joined the organization. |
| DateofTermination | Date | Date the employee left the organization (if applicable). |
| TermReason | Text | Reason for employee termination. |
| EmploymentStatus | Text | Current employment status of the employee. |
| Department | Text | Department where the employee works. |
| ManagerName | Text | Name of the employee's reporting manager. |
| ManagerID | Integer | Unique identifier of the reporting manager. |
| RecruitmentSource | Text | Source through which the employee was recruited. |
| PerformanceScore | Text | Employee's latest performance rating. |
| EngagementSurvey | Decimal | Employee engagement survey score. |
| EmpSatisfaction | Integer | Employee satisfaction rating. |
| SpecialProjectsCount | Integer | Number of special projects completed by the employee. |
| LastPerformanceReview_Date | Date | Date of the employee's most recent performance review. |
| DaysLateLast30 | Integer | Number of times the employee was late during the last 30 days. |
| Absences | Integer | Total number of employee absences. |
